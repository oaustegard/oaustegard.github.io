at-protocol-extension

A simple Chrome Extension that provides a context menu that allows opening a selected at: protocol URI in bsky.app

For example, applying it to at://did:plc:r2whjvupgfw55mllpksnombn/app.bsky.feed.post/3lhm2av677s2x would open https://bsky.app/profile/did:plc:r2whjvupgfw55mllpksnombn/post/3lhm2av677s2x

View the project on GitHub: https://github.com/oaustegard/at-protocol-extension/tree/main
